# Whiskey Cat Hotel


## Frontend:
Located in `./app`

### To start in dev:
`cd app`
`pnpm install`
`pnpm run dev`

## Backend (API):
Located in `./api`

### To start in dev:
`cd api`

`pnpm install`

`pnpm seed-admin && pnpm seed-rooms` *(This only needs to be ran once, to seed the admin user)*

`pnpm run dev`
